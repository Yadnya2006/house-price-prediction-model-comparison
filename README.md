# house-price-prediction-model-comparison

## Overview

This project compares multiple Machine Learning regression models on the California Housing Dataset to identify the best-performing algorithm for house price prediction.

## Models Used

* Linear Regression
* Ridge Regression
* Decision Tree Regressor

## Data Preprocessing

* Feature Scaling using StandardScaler
* Train-Test Split (80:20)

## Evaluation Metrics

| Model             | RMSE     | R² Score |
| ----------------- | -------- | -------- |
| Linear Regression | 0.745581 | 0.575788 |
| Ridge Regression  | 0.745554 | 0.575819 |
| Decision Tree     | 0.724234 | 0.599732 |

## Best Model

Decision Tree Regressor

* RMSE: 0.724234
* R² Score: 0.599732
